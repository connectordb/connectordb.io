Interpolators

Interpolators form the core of PipeScript's dataset generation capabilities. Unlike transforms, which are functions, and accept arguments, interpolators are simple identifiers. They do not accept arguments.

You can also use PipeScript instead of an interpolator. Simply in place of your interpolator, put a script. This is a useful stopgap for when the interpolator you desire is not available. Note that PipeScript will return null when there are no datapoints in an interpolation period.

If you want to generate datasets using ConnectorDB, the python documentation should get you started easily.
